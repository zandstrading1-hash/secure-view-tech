# Secureview Tech — Website

Static site (plain HTML/CSS/JS). No build step. Open `index.html` in any browser to view.

## See it live for free (GitHub Pages)
1. Create a new GitHub repo and push these files to the **root** (index.html must be at the top level).
2. Repo → **Settings** → **Pages**.
3. Under "Build and deployment", Source = **Deploy from a branch**, Branch = **main** / **/(root)**, Save.
4. Wait ~1 minute. Your site is at `https://<your-username>.github.io/<repo-name>/`.
