# Secureview Tech launch handoff

The website code is launch-ready. These account-owner actions complete services that cannot be activated from the repository alone:

1. Submit the estimate form once and open the activation email sent by FormSubmit to `secureview.tech@gmail.com`.
2. If a custom domain is purchased, point its DNS to GitHub Pages, add the domain in the repository Pages settings, then update canonical, sitemap, form-return, and social-image URLs.
3. Add the final custom domain to Google Search Console and submit `/sitemap.xml`.
4. Replace the Gmail address only after a domain mailbox has been created and tested.
5. Have the published privacy notice, terms, and project agreement reviewed for the business's actual practices.
